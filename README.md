# Image Encryption Tool

## Description
The Image Encryption Tool is a cybersecurity project developed using Python and Tkinter. This tool helps secure image files through encryption techniques, protecting visual data from unauthorized access and tampering.

Users can encrypt and decrypt images using a secret key. The project demonstrates basic image security concepts and data protection methods in cybersecurity.

## Features
- Encrypt image files
- Decrypt encrypted images
- Secret key protection
- User-friendly graphical interface
- Supports PNG and JPG images

## Technologies Used
- Python
- Tkinter
- Pillow Library
